# 🛠️ Ostrascript Support

Nejostřejší rozšíření pro nejostřejší jazyk na světě. 
Tento plugin přidává podporu pro syntaxi a barvičky pro soubory `.os`.

## ✨ Funkce
* **Syntax Highlighting**: Všechna klíčová slova (`kalfas`, `pyco`, `toz`) jsou krásně barevná.
* **Auto-closing**: Automatické doplňování závorek a uvozovek.
* **Ostravský styl**: Programování bez zbytečných keců.
